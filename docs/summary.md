# Data Ecosystem Summary

This file provides a high-level overview of the assessment data files available in the `/data` directory. For a highly detailed breakdown of every single column, field, and mapping, please refer to **`CONTEXT.md` (Section 5 & Appendix A)**.

## 1. Mentor Assessments
**File:** `Year 1 Assessment_Matrix (1) (1) (1).xlsx`
- Contains tabs for `Kickstart`, `Legacy`, `Murder Mystery` (Marketing), `Business Xray`, `SDP`, and `Accounts`. 
- Rows represent the 24 readiness sub-parameters, and columns represent the 17 students. Values are 1-10 scores.
